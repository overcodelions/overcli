# Code of Conduct

Overcli is a small project and a father–son collaboration. We expect everyone interacting with the project — issues, PRs, discussions, anywhere else — to be respectful and patient.

## In short

- Be kind and assume good faith.
- Disagree about ideas; never about people.
- No harassment, slurs, personal attacks, or sexualised language.
- If you wouldn't say it to a colleague's face, don't post it here.

## If something goes wrong

Email **assistant@codelionsllc.com** with a description of what happened and links if relevant. Reports will be read promptly and handled privately.

The maintainers may, at their discretion, edit or remove comments, close issues or PRs, and ban people from the project for behaviour that violates this code.

This document is loosely based on the [Contributor Covenant](https://www.contributor-covenant.org/), trimmed to fit a small project.
